Angular ======= Angular is a JavaScript Framework which allows us to create reactive Single-Page-Applications (SPAs). npm install -g angular-cli npm install -g angular-cli@latest npm uninstall -g angular-cli ng -v (Check angular and node.js version) Preparation: 1. Install: node.js 2. Install Angular CLI globally: npm install -g @angular/cli 3. cd applicationFolderPath (where new folder will be created) 4. ng new appName 5. cd applicationFolderPath (newly created folder path) 6. ng serve Installing bootstrap: 1. cd applicationFolderPath 2. npm install --save bootstrap
