Angular ======= Angular is a JavaScript Framework which allows us to create reactive Single-Page-Applications (SPAs). npm install -g angular-cli npm install -g angular-cli@latest npm uninstall -g angular-cli ng -v (Check angular and node.js version) Preparation: 1. Install: node.js 2. Install Angular CLI globally: npm install -g @angular/cli 3. cd applicationFolderPath (where new folder will be created) 4. ng new appName 5. cd applicationFolderPath (newly created folder path) 6. ng serve Installing bootstrap: 1. cd applicationFolderPath 2. npm install --save bootstrap Importing bootstrap: .angular-cli.ison > styles > (add the following:) "../node_modules/bootstrap/dist/css/bootstrap.min.css", Result: "styles": [ "../node_modules/bootstrap/dist/css/bootstrap.min.css", "styles.css" ], Create a new component by hand ------------------------------ 1. Create folder for example: server 2. in this folder cretae a ts file with the same name as folder - server.component.ts 3. in this folder create a html file with the same name as folder - server.component.html 4. In server.component.ts add: import {Component} from '@angular/core'; @Component({ selector: 'app-server', //any uniq name for selector templateUrl: './server.component.html' //path to the html file }) export class ServerComponent { } 5. add some info in - server.component.html
