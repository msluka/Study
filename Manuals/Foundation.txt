http://foundation.zurb.com/sites/docs/installation.html

1. Install nodejs - https://nodejs.org/en/download/

2. run Power Shell (in admin mode)

3. npm install --global foundation-cli



4. foundation new



5. choose: A website (Foundation for Sites)

  

6. set project-name

7. Basic Template: includes a Sass compiler



8. cd project-name folde path

9. npm start
